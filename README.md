# Parallel Systems Lab (PSL) @ NTU

We build systems infrastructure at the intersection of parallel computing
and AI — spanning high-performance I/O, parallel file sytems, GPU kernel optimization, LLM optmization, and agentic memory systems.

## Research Areas

**Parallel I/O & Storage** — High-performance data pipelines; storage consistency semantics;
parallel file systems; I/O for large-scale HPC and AI workloads.

**AI Systems & GPU Optimization** — Efficient training and inference
infrastructure; kernel optimization; memory and compute scheduling.

**Agentic AI Systems** — Systems support for multi-agent workflows;
orchestration; efficient memory operations.
